Album Detective

Discover which albums you own across your collections

Album Detective helps music collectors identify their discography across local and online music libraries. Compare your physical collection against Spotify, Jellyfin, and other platforms to find gaps and overlaps.

What it does

  • Scans local music folder in read-only mode
  • Multi-threaded indexing for faster scans
  • Smart caching to avoid re-indexing unchanged files
  • Supports multiple scan profiles and worker configuration
  • Extracts metadata or uses folder structure fallback
  • Compares local library vs online platforms
  • Exports missing albums and artists
  • Interactive UI with file explorer and comparison tables
  • Supports Spotify API import and Jellyfin integration
  • Incremental NAS import with local caching
  • 14 curated theme palettes
  • CSV viewer with sortable headers and search filtering
  • Artwork resolution via MusicBrainz and Cover Art Archive
